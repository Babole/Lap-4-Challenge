# Lap-4-Challenge
### Installation & Usage 
----
#### Installation
- Fork and clone the repo
- `pipenv shell`
- Run `pipenv install` in the terminal to install dependencies
- `python app.py` or `pipenv run dev` to start the server

#### Usage
- Head to http://127.0.0.1/:5000 or `localhost:5000/urls`
